<h2 align="center">Interviews</h2>


A service that allows users to share their impressions of the interviews conducted


### Development tools

**stack:**

- Python
- FastAPI
- Postgres
- Celery
- Redis

## Start

#### 1) Create image

    docker-compose build

##### 2) Run container

    docker-compose up

##### 3) Go to address

    http://127.0.0.1:8000/api/v1/swagger/
